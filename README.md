# Habit Tracker

A simple, privacy-first habit tracking application built with Flutter. Track your daily habits, build streaks, and improve your life one day at a time.

## Features

- 📝 **Create Habits** - Add custom habits with custom colors
- ✅ **Daily Tracking** - Mark habits as complete each day
- 🔥 **Streaks** - See your current streak for each habit
- 🔒 **Privacy First** - All data stored locally on your device
- 📱 **Offline** - Works completely offline, no internet required
- 🎨 **Beautiful UI** - Clean Material Design 3 interface

## Privacy

This app is designed with your privacy in mind:
- **No accounts required**
- **No data collection**
- **No analytics or tracking**
- **100% offline storage**

View our [Privacy Policy](https://yourusername.github.io/habit-tracker/privacy-policy.html) for more details.

## Getting Started

### Prerequisites

- Flutter SDK (>=3.0.0)
- Android SDK

### Installation

1. Clone the repository
2. Get dependencies: `flutter pub get`
3. Run the app: `flutter run`

### Building

```bash
# Debug APK
flutter build apk --debug

# Release APK
flutter build apk --release
```

## GitHub Pages

This project includes a `docs/` folder for GitHub Pages deployment. To deploy:

1. Go to your repository settings on GitHub
2. Navigate to "Pages" section
3. Select "Deploy from a branch"
4. Choose `main` branch and `/docs` folder
5. Save

Your app's landing page will be available at:
`https://yourusername.github.io/habit-tracker/`
