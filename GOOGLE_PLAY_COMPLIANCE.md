# Google Play Compliance Pack - Habit Tracker

## App Concept Summary
Habit Tracker is a completely offline application that helps users build and maintain daily habits. Users can create custom habits with different colors, track their daily completion, and view their streak progress. All data is stored locally using SQLite database.

---

## Privacy Policy

### Habit Tracker - Privacy Policy

**Last Updated: 2026**

#### 1. Data Collection
Habit Tracker does NOT collect, store, transmit, or share any personal user data. The app operates completely offline and all habit data is stored solely on the user's device.

#### 2. Local Storage
- All habits (name, color) and completion logs are stored locally on the device using SQLite database
- No cloud storage or synchronization
- No backup to external servers

#### 3. Third-Party Services
This app does NOT use any third-party analytics services, advertising networks, or data collection tools.

#### 4. Permissions
This app requires minimal permissions:
- Storage access (only for local database operations on some Android versions)

#### 5. Children's Privacy
This app is safe for users of all ages. It does not collect any personal information from children or adults.

#### 6. Changes to Privacy Policy
As this app does not collect any data, there will be no changes to data collection practices.

#### 7. Contact
For any questions about this privacy policy, please contact the developer through the app store support channels.

### Full Description

# Habit Tracker

**Build better habits, one day at a time**

Habit Tracker is a completely free, privacy-focused application that helps you build and maintain positive daily habits. All your progress is stored securely on your device.

## Features

✓ **Custom Habits** - Create unlimited habits with custom colors
📊 **Daily Tracking** - Mark habits as complete each day
🔥 **Streak Tracking** - Track your consecutive day streaks
🎨 **Color Coding** - Personalize each habit with a unique color
🔒 **Privacy First** - Works 100% offline, no data collection

## Why Use Habit Tracker?

✓ **100% Offline** - Works without internet connection
✓ **No Login Required** - Start using immediately
✓ **No Ads** - Clean, distraction-free experience
✓ **No Data Collection** - Your privacy is protected
✓ **No Cloud Sync** - All data stays on your device
✓ **Build Habits** - Track your daily progress toward goals

## How to Use

1. Tap the + button to create a new habit
2. Give your habit a name and choose a color
3. Tap a habit each day to mark it complete
4. Build your streak by completing habits daily
5. Long press to delete a habit
